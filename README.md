# Data-Pre-processing-and-Visualization

📌 Overview
This project focuses on transforming a large-scale raw e-commerce dataset into a clean, structured, and analysis-ready dataset using comprehensive data preprocessing techniques.
The dataset used is Pakistan’s Largest E-Commerce Dataset, containing over 1 million transaction records, which required extensive cleaning and transformation before meaningful analysis could be performed.
The project demonstrates the end-to-end data preparation pipeline, from raw data exploration to final analytical outputs using SAS and Power BI.

⚙️ Tools & Technologies
- SAS Studio – Data cleaning, transformation, and preprocessing
  Using PROC SQL language which is a procedure within the SAS programming language that implements the Structured Query Language (SQL)       standard.
- Power BI – Data visualization and dashboarding
- Excel/CSV – Data cleaning, storage, and handling

📈 Key Results
- Dataset reduced from 1,048,587 → 176,932 records (high-quality data)
- Significant improvement in data distribution and consistency
- Enabled accurate trend analysis and visualization
- Prepared dataset suitable for machine learning and predictive modeling
